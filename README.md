# Simple-Console-Game-Using-C-lang
Simple Console Game using C lang



# Features

1- Roles System (2 roles till now (Owner,Player)

2- Monster system 

3- fighting system

4- All the data is stored into txt files

5- Admin Panel

6- Shop system

7- Status System
